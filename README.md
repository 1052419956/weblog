WeBlog
======

A simple blog system based on [Flask](http://flask.pocoo.org/)


Quick Start
-----------

````
$ git clone https://github.com/ghostrong/weblog.git
$ cd weblog
$ pip install -r requirements.txt
$ python run.py
````

Now, visit `http://127.0.0.1:8888` in a browser.
